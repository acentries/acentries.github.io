GBookmark

Google Bookmarks Client

GBookmark is Google Bookmarks client for Android.

It help you to use Google Bookmarks, that is online bookmark service provided by Google.

Google Bookmarks: https://www.google.com/bookmarks/

Feature

  • Auto Synchronization.
  • Add, Edit, Delete Bookmark.
  • Create Desktop Shortcut.
  • Search bookmark from title,label,URL and note.

  • Support label separator for folder view.

  • Create Bookmark by URL sharing from WEB Browser.

  • Support Favicon download.

Requirements

Android 2.3 later
